# AI Feedback Analyzer

An AI-powered platform that analyzes customer reviews using Groq LLM.

## Features

- CSV Upload
- Sentiment Analysis
- Category Detection
- AI Summary Generation
- Interactive Dashboard
- PDF and CSV Report Export

## Tech Stack

- Streamlit
- Python
- Pandas
- Groq LLM
- Plotly
- ReportLab

## Run

pip install -r requirements.txt

streamlit run app.py
